Welcome to Webix

Webix is a multi-widget JavaScript UI library for developing HTML5 and CSS3 compatible mobile and desktop web apps. It provides more than 88 rich UI widgets and controls for data management, visualization, uploading, building layouts and editing.

Using these components, you can build web apps that will run properly not only on personal computers but also on iOS, Android and Windows touch devices. Webix offers simple JQuery, Angular, Vue.js, React and Backbone integration and can work with any server-side platform, such as PHP, ASP.NET, Java, Ruby, etc.
